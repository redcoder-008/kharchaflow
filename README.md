# 💸 KharchaFlow

**A modern, fast, and intuitive expense tracker built to help you manage your finances with ease.**

🌐 **Live Demo:** https://kharchaflow.vercel.app

---

## ✨ Features

* 💰 Income & Expense Tracking
* 📊 Interactive Dashboard & Analytics
* 📅 Nepali & English Calendar Support
* 🔔 In-App & Push Notifications
* 🗑️ Recycle Bin for Deleted Transactions
* 📈 Monthly Reports & Insights
* 👤 User Authentication & Profile Management
* 🛠️ Admin Dashboard
* 📱 Responsive Design
* 🌙 Dark & Light Theme

---

## 🛠️ Tech Stack

* **Frontend:** React, TypeScript, Tailwind CSS
* **Backend:** Firebase
* **Hosting:** Vercel

---

## 🚀 Getting Started

```bash
git clone https://github.com/your-username/kharchaflow.git
cd kharchaflow
npm install
npm run dev
```
